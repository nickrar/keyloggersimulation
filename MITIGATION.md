### Keylogger Mitigation Techniques (Lesson)

1. **Use keyboard encryption software**

   * Keyboard encryption can help protect typed input by encrypting keystrokes before they are exposed to keyloggers.
   * This makes the captured keystroke data harder to read or less useful to attackers.

2. **Keep Windows and security software updated**

   * Users should always update Windows to the latest version.
   * Security updates fix weaknesses that attackers may use to install malware or keyloggers.

3. **Avoid downloading suspicious software**

   * Users should avoid cracked software, unknown installers, suspicious email attachments, and files from untrusted websites.
   * Many keyloggers infect devices because users accidentally install unsafe programs.

4. **Use MFA and password managers**

   * Multi-factor authentication adds another layer of protection if a password is captured.
   * Password managers reduce the need to manually type passwords, which lowers the chance of passwords being captured through keystrokes.
